# 云启 · 新标签页

一款个性化浏览器新标签页扩展，提供时钟、多引擎搜索、快捷链接、备忘笔记、书签管理等组件，支持组件外观自定义、WebDAV 云同步与加密备份。

## 功能特性

### 新标签页

- **🕐 时钟组件**：实时时间与日期显示，可自定义数字大小和粗细
- **🔍 搜索组件**：支持多搜索引擎切换，自定义下拉框样式，回车即搜
- **🔗 快捷链接**：常用网站快速访问，支持拖拽排序，自定义字体大小粗细
- **📝 备忘笔记**：全屏编辑模式，支持搜索过滤和分页浏览，自动保存，万级笔记不卡顿
- **📚 书签组件**：浏览器书签树展示，支持折叠展开，懒加载优化
- **🎨 外观自定义**：每个组件独立设置背景色、透明度、圆角、内边距、文字颜色、字体大小/粗细等
- **🖼️ 背景设置**：支持图片/视频/纯色背景，可切换图片与视频模式
- **📐 组件排序**：设置面板内拖拽排序组件显示顺序
- **👁️ 组件开关**：独立控制每个组件的显示与隐藏
- **🖱️ 操作按钮**：全局开关控制组件上的操作按钮显隐

### WebDAV 云同步

- **配置同步**：新标签页的所有配置（组件布局、外观、快捷链接、笔记等）通过 WebDAV 云端备份
- **书签备份**：将浏览器书签导出并上传至 WebDAV 服务器
- **版本轮转**：自动保留 2 份备份（最新 v1 + 历史 v2），旧备份自动覆盖
- **加密存储**：可选 AES-256-GCM 加密，PBKDF2 密钥派生，防篡改防泄露
- **双向同步**：支持从 WebDAV 拉取书签合并到本地，已存在则跳过，不存在则创建
- **右键菜单**：页面右键即可快速备份/恢复/切换新标签页

### 其他

- **跨浏览器**：兼容 Chrome、Edge、Firefox（Manifest V3）
- **性能优化**：懒加载、DocumentFragment 批量渲染、requestIdleCallback 空闲加载
- **自定义弹窗**：所有确认操作使用自定义弹窗，无原生 alert/confirm

## 安装方式

### Chrome / Edge

1. 下载本项目源码
2. 打开浏览器扩展管理页：
   - Chrome：`chrome://extensions/`
   - Edge：`edge://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本项目目录
5. 安装完成，打开新标签页即可使用

### Firefox

1. 打开 `about:debugging#/runtime/this-firefox`
2. 点击「临时加载附加组件」
3. 选择项目中的 `manifest.json` 文件
4. 安装完成

## 使用指南

### 新标签页

打开新标签页后，默认显示时钟、搜索、快捷链接三个组件。点击右上角 ⚙️ 图标进入设置面板：

| 设置项     | 说明                                      |
| ------- | --------------------------------------- |
| 组件开关    | 独立控制时钟、搜索、快捷链接、备忘笔记、书签的显示与隐藏            |
| 组件排序    | 拖拽调整组件在新标签页上的显示顺序                       |
| 组件外观    | 每个组件独立设置背景色、透明度、圆角、内边距、文字颜色、字体大小/粗细等    |
| 背景设置    | 设置新标签页背景（图片URL/视频URL/纯色），可切换图片与视频模式     |
| 操作按钮开关  | 控制组件上的操作按钮（新建、删除等）是否显示                  |
| 快捷链接管理  | 添加、编辑、删除快捷链接，拖拽排序                       |
| WebDAV  | 配置 WebDAV 连接，备份/恢复新标签页配置和书签             |

### 组件外观自定义

每个组件都有独立的外观参数：

| 参数     | 说明                    | 范围          |
| ------ | --------------------- | ----------- |
| 背景色    | 组件卡片背景颜色              | 颜色选择器       |
| 透明度    | 0 = 无背景，1 = 完全不透明     | 0 ~ 1       |
| 圆角     | 卡片圆角大小                | 0 ~ 40px    |
| 内边距    | 卡片内边距                 | 0 ~ 60px    |
| 文字颜色   | 组件文字主色                | 颜色选择器       |
| 字体大小   | 内容文字大小                | 10 ~ 28px   |
| 字体粗细   | 内容文字粗细                | 100 ~ 900   |
| 标题字体大小 | 标题文字大小                | 10 ~ 36px   |
| 标题字体粗细 | 标题文字粗细                | 100 ~ 900   |

部分组件有特有参数：

| 组件 | 特有参数 | 说明 |
| --- | ----- | --- |
| 时钟 | 数字大小 | 时钟数字字号（24 ~ 120px） |
| 时钟 | 数字粗细 | 时钟数字字重（100 ~ 900） |
| 搜索 | 下拉框背景色 | 搜索引擎下拉框背景颜色 |
| 搜索 | 下拉框文字色 | 搜索引擎下拉框文字颜色 |

### WebDAV 配置

点击扩展图标或设置面板中的 WebDAV 区域，填写：

| 配置项     | 说明                                                |
| ------- | ------------------------------------------------- |
| WebDAV地址 | WebDAV 服务器地址，如 `https://dav.example.com/dav/user` |
| 用户名     | WebDAV 认证用户名                                      |
| 密码      | WebDAV 认证密码                                       |
| 备份目录    | WebDAV 上的备份目录路径，默认 `/bookmark-backup`             |

> 常见 WebDAV 服务：坚果云（国内推荐）、Nextcloud/ownCloud（自建）、Alist（聚合网盘）

### 加密设置（可选）

在设置面板中勾选「启用加密」并设置加密密码。启用后，备份数据将以 AES-256-GCM 加密存储到 WebDAV，恢复时需使用相同密码解密。

> 请牢记加密密码，丢失后将无法恢复备份数据。

### 备忘笔记

- 列表视图显示笔记摘要卡片（第一行标题 + 第二行摘要）
- 点击卡片进入全屏编辑模式，支持多行长文本
- 编辑时每 2 秒自动保存，失焦时立即保存
- 支持搜索过滤（300ms 防抖）和分页浏览（每页 10 条）
- 新建笔记后直接进入编辑器

### 右键菜单

☁️ 云启 · 新标签页

  ├── ⬇️ 恢复历史备份(v2)

### 组件外观系统

采用 CSS 自定义属性（CSS Variables）实现组件级样式控制：

```
applyComponentStyle(wrapper, componentKey)
  ├── 读取 getComponentStyle(key) 获取合并后的样式配置
  ├── 计算 hexToRgb() 颜色转换
  ├── 设置 CSS 变量到 wrapper 元素
  │   ├── --comp-glass-bg / --comp-glass-border（背景/边框）
  │   ├── --comp-text-primary / --comp-text-secondary（文字颜色）
  │   ├── --comp-font-size / --comp-font-weight（字体）
  │   ├── --comp-glass-radius / --comp-padding（圆角/内边距）
  │   └── 组件特有变量（时钟数字、搜索下拉框等）
  └── CSS 选择器根据 data-component 属性应用特有样式
```

### 加密方案

| 组件   | 实现                                           |
| ---- | -------------------------------------------- |
| 加密算法 | AES-256-GCM（认证加密，防篡改）                        |
| 密钥派生 | PBKDF2（SHA-256，100,000 次迭代）                  |
| Salt | 每次加密随机生成 16 字节                               |
| IV   | 每次加密随机生成 12 字节（NIST 推荐）                      |
| 存储格式 | Base64(salt\[16] + iv\[12] + 密文 + 认证标签\[16]) |
| 加密标识 | `ENC1:` 前缀，恢复时可靠判断数据是否加密                     |

### 备份策略

采用**轮转槽位策略**，始终保留 2 份备份：

```
备份时：
  1. 如果 v1 存在，将 v1 移动到 v2（覆盖旧 v2）
  2. 将当前数据导出并上传为 v1

恢复时：
  用户可选择从 v1（最新）或 v2（历史）恢复
```

### 性能优化

| 优化项              | 实现方式                                         |
| ---------------- | -------------------------------------------- |
| 书签懒加载            | requestIdleCallback 空闲时渲染书签树                  |
| 笔记分页             | 每页仅渲染 10 条，万级笔记不卡顿                           |
| 批量 DOM 插入        | DocumentFragment 减少 reflow                   |
| 设置面板延迟初始化        | 首次点击时才创建，减少首屏负担                              |
| 搜索防抖             | 300ms 防抖，避免频繁过滤渲染                            |
| Favicon 缓存       | Chrome 内置 _favicon API + 内存缓存                |
| 自动保存节流           | 笔记编辑 2 秒定时保存，避免每次按键都写存储                      |

## 注意事项

1. **CORS 问题**：部分 WebDAV 服务器未配置跨域资源共享（CORS），可能导致浏览器请求被拒绝。请确保服务器允许跨域访问。
2. **密码安全**：WebDAV 密码和加密密码存储在浏览器本地存储中，建议在个人设备上使用。
3. **书签去重**：恢复时以 URL 为唯一标识进行去重，同一 URL 的不同标题书签会被跳过。
4. **大书签文件**：书签数量极大时（>5000），备份和恢复可能需要较长时间，请耐心等待。
5. **新标签页开关**：右键菜单可切换新标签页功能，关闭后打开新标签页将显示空白页。
